## Инструкция по запуску api
1. Для запуска нужна версия python 3.10.
2. Запустить в терминале pip install -r requirements.txt из директории Hackathon-2022/stratagems.
3. Запустить в терминале python manage.py runserver из директории Hackathon-2022/stratagems и открыть http://127.0.0.1:8000.
